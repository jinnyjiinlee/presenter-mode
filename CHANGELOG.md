# 변경 이력 (Changelog)

발표자 모드의 버전별 변경 사항이에요. 최신 버전이 맨 위에 있어요.

## v5.2 — 2026-07-12

**쉬는화면을 코드 수정 없이 누구나 꾸밀 수 있게**

- ☕ **쉬는화면 설정 UI** — 발표자 창 하단 [☕ 쉬는화면] 버튼 → 제목·부제·로고(텍스트/이미지)·QR 이미지·시계·BGM을 폼에서 설정. 이미지는 파일 선택만 하면 돼요 (브라우저에 덱별 저장, 코드 수정 없음)
- **상황별 프리셋 6종** — ⬛ 검은 화면(기본) · ☕ 쉬는 시간 · 🍽 점심시간 · 💻 실습 시간 · 🙋 Q&A · 🕘 곧 시작. 클릭 한 번으로 바로 적용
- UX 다듬기 9종 — 판서 버튼 활성 상태 표시("✏️ 판서 중") · 판서 툴바 단축키 뱃지 · 지우개 커서 구분 · 확대 해제 안내 토스트 · 온보딩 배너에 새 기능 안내 · 발표자 창에서 다음 슬라이드 박스 클릭 = 이동 · 적용 중 프리셋 하이라이트 · 발표자 창 판서 툴바 겹침 방지
- 버그 수정: 돋보기(M)를 한 번 쓰면 덱의 등장 애니메이션이 꺼지던 문제

## v5.1 — 2026-07-12

**구역 확대 + 가리기 기본값 정리**

- 🔎 **Z 구역 확대** — 드래그한 영역을 화면에 꽉 채워 확대 (클릭=2배, 배율 1.15~5). 발표자 창의 현재 슬라이드에서 드래그해도 강의장 화면이 확대돼요. 다시 Z·Esc·슬라이드 넘기면 원위치
- **B 가리기 기본값 = 순수 검은 화면** — 쉬는화면 꾸미기는 옵트인 (템플릿 기본값에서 제작사 브랜딩 제거)
- M 돋보기를 발표자 창에서 누르면 사용법 안내 토스트 표시

## v5 — 2026-07-12

**커뮤니티 피드백 반영 — 판서 · 쉬는화면 · BGM · 돋보기**

- ✏️ **판서** — `D` 펜 / `G` 형광펜 / `E` 지우개, 색상 6종. 발표자 창·강의장 화면 어디서 그려도 실시간 동기화 (슬라이드별 유지, 새로고침 시 초기화)
- ☕ **쉬는 시간 화면** — `B` 가리기를 로고·안내 문구·QR·현재 시각이 뜨는 안내 화면으로 커스터마이즈 (`BREAK_SCREEN` 설정)
- ♪ **쉬는 시간 BGM** — mp3 지정 시 쉬는 시간에 재생, ♪ 버튼으로 재생/정지
- 🔍 **M 마우스 돋보기** — 커서 주변을 원형 렌즈로 확대, 휠로 배율 1.5~4배
- 버그 수정: 창 너비가 1600px보다 좁을 때(1440×900 맥북 풀스크린 포함) 슬라이드가 중앙에서 벗어나고 오른쪽이 잘리던 문제

## v2~v3 — 2026-07-12

**실전 개선 60종** (v2 30개 + v3 30개 — 실제 7시간 풀데이 강의에서 나온 요구사항)

- 진행: 리모컨(클리커) 호환, 숫자+Enter 점프, Home/End, 마지막 장에서 안 튐(랩어라운드 제거)
- 발표자 창: 필름스트립(썸네일 크기 조절·노트 배지), 현재/다음 슬라이드 비율 드래그 조절, 노트 크기 조절·접기, 다음 노트 미리보기, 노트 큐 자동 강조(⏰주황 ⚠️빨강 백업파랑), 체류 시간, 카운터 클릭=남은 장수
- 타이머: 목표 시간·남은 시간 칩·5분 전/초과 알림, 창 닫아도 유지, 리셋 2번 확인
- 화면: B/W 블랙아웃(슬라이드 넘기면 자동 해제), F/F5 전체화면, Wake Lock, 4초 아이들 시 커서·힌트 숨김
- 연결: 2초 하트비트 연결 상태 칩(●/○), 어긋난 슬라이드 자동 재정렬, 발표 중 강의장 화면에 도움말·점프창 노출 차단
- 라이트/다크 테마 토글, 첫 실행 온보딩, ? 단축키 도움말, 팝업 차단 원클릭 재시도
